+ 查找替换
```shell
# 当前文件查找
ctrl + F

# 在项目进行查询
ctrl + shift + F

# 当前文件替换
ctrl + R

# 在项目中进行替换
ctrl + shift + R

# 查找 class
ctrl + N

# 查找文件
ctrl + shift + N
```

+ 切换跳转
```shell
# 最近更新的代码
ctrl + E

# 跳转历史光标
ctrl + alt + ←/→

# 跳转到行
ctrl + G
```
